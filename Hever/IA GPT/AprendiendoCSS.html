Introducción a CSS

¿Qué es CSS?

CSS (Cascading Style Sheets) es un lenguaje de estilos utilizado para definir la presentación de documentos HTML. Controla colores, fuentes, disposición, y otros aspectos visuales de una página web.

Historia y evolución de CSS

CSS fue propuesto por Håkon Wium Lie en 1994 y estandarizado por el W3C en 1996 con CSS1. CSS2 (1998) y CSS3 (modular desde 1999) introdujeron nuevas características como animaciones y layouts avanzados.

Ventajas de usar CSS

CSS permite separar el contenido (HTML) del diseño, mejora la accesibilidad, reduce el tiempo de carga, y facilita el mantenimiento de sitios web.
