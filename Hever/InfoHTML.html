¿Qué es HTML?

HTML (HyperText Markup Language) es el lenguaje que se utiliza para estructurar el contenido en la web. Utiliza etiquetas para definir elementos como títulos, párrafos, enlaces, imágenes, listas y más.

Historia

HTML fue creado por Tim Berners-Lee en 1991 como una forma de compartir documentos científicos. Desde entonces ha evolucionado en múltiples versiones.

Versiones

  • HTML 1.0: Primer borrador no oficial (1991)
  • HTML 2.0: Estándar oficial en 1995
  • HTML 4.01: Introdujo mejoras importantes (1999)
  • HTML5: Versión actual y más utilizada (2014)

Estructura Básica

Todo documento HTML debe seguir una estructura base:

<!DOCTYPE html>
<html>
  <head>
    <title>Título de la página</title>
  </head>
  <body>
    Contenido visible de la página
  </body>
</html>

Elementos de Texto

  • <h1> a <h6>: Encabezados
  • <p>: Párrafos
  • <strong>, <em>: Negrita y cursiva

Imágenes y Multimedia

  • <img src="imagen.jpg" alt="Descripción">: Inserta una imagen
  • <audio> y <video>: Para contenido multimedia

Listas y Tablas

  • <ul>: Lista desordenada
  • <ol>: Lista ordenada
  • <table>: Tabla con filas y columnas
