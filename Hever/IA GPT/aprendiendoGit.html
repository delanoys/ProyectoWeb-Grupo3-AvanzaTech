¿Qué es Git?

Git es un sistema de control de versiones distribuido que permite a múltiples desarrolladores trabajar en el mismo proyecto de forma eficiente.

Comandos Básicos de Git

  • git init: Inicializa un nuevo repositorio.
  • git status: Muestra el estado del repositorio.
  • git add .: Añade archivos al área de preparación.
  • git commit -m \"mensaje\": Crea un commit.
  • git log: Muestra el historial de commits.

Repositorios

Un repositorio es donde se almacenan los archivos y el historial de versiones. Puede ser local (en tu máquina) o remoto (en servidores como GitHub).

Ramas (Branches)

Las ramas permiten trabajar en nuevas funcionalidades sin afectar la versión principal. El comando git branch gestiona ramas.

¿Qué es GitHub?

GitHub es una plataforma de alojamiento de repositorios Git que facilita la colaboración y publicación de código de forma remota.
