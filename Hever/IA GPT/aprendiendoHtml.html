Introducción a HTML

¿Qué es HTML?

HTML (HyperText Markup Language) es el lenguaje estándar para crear páginas web. Permite estructurar el contenido mediante el uso de etiquetas.

Historia y evolución de HTML

HTML fue desarrollado a finales de los 80s por Tim Berners-Lee. Desde entonces ha evolucionado hasta HTML5, la versión actual que permite integrar multimedia, gráficos y más.

Estructura Básica

Etiquetas fundamentales

Las etiquetas básicas incluyen <html>, <head>, <title>, <body>, entre otras.

Comentarios y buenas prácticas

Los comentarios en HTML se escriben así: <!-- Esto es un comentario -->. Es buena práctica escribir código claro y bien indentado.

Elementos de Texto

Títulos y párrafos

Los títulos se definen con <h1> a <h6> y los párrafos con <p>.

Formato de texto

HTML permite aplicar formato con etiquetas como <strong>, <em>, <mark> y más.

Imágenes y Multimedia

Insertar imágenes

La etiqueta <img> se usa para mostrar imágenes. Ejemplo: <img src="imagen.jpg" alt="Descripción">.

Enlaces, video y audio

Los enlaces usan <a>, el video se inserta con <video> y el audio con <audio>.

Listas y Tablas

Listas ordenadas y no ordenadas

Las listas ordenadas usan <ol> y las no ordenadas <ul>. Los elementos de lista son <li>.

Tablas básicas

Las tablas usan <table> con filas <tr>, encabezados <th> y celdas <td>.
